# Omni Beam Former

<BlockImage id="me_beam_former:omni_beam_former_block" scale="6" p:facing="north" />

A multi-link beam device. Use the Laser Binding Tool to link it to other Omni Beam Formers.

## Range

- Horizontal range: within 16 blocks (X/Z).
- Vertical range: within 32 blocks (Y).
