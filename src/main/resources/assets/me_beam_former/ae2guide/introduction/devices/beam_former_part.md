# Beam Former (Part)

<ItemImage id="me_beam_former:beam_former_part" scale="4" />

A cable part that forms a point-to-point AE2 connection using a beam.

## Usage

- Place it on an AE2 cable.
- Point it at another Beam Former (Part) facing back.
- A solid block blocks the beam.
- Effective range: up to 32 blocks.

## Toggle beam

- Sneak + right-click with empty hand to hide/show the beam.
