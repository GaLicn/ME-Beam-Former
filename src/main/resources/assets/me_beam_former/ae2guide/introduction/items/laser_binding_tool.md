# Laser Binding Tool

<ItemImage id="me_beam_former:laser_binding_tool" scale="4" />

Used to bind Omni Beam Formers and Wireless Energy Towers.

## Controls

- Sneak + right-click: select a source (tower / omni).
- Right-click: link/unlink (tower-to-tower or omni-to-omni).
- Sneak + left-click (in air): clear the selected source.
